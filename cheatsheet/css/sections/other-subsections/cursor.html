Переменные

cursor: auto;            (defaulf)
      - default;
      - pointer;
      - text;
      - progress;
      - wait;
      - help;
      - crosshair;
      - not-allowed;
      - url(../..), alt;
      - none;

default - стрелка
pointer - палец
text - текстовый
progress - стрелка с вращающимся индикатором процесса
wait - вращающийся индикатором процесса
help - стрелка со знаком вопроса
crosshair - перекрестие
not-allowed - перечёркнутый красный круг
url - из файла, или alt - один из стандартных, если не загрузится
none - не отображается
