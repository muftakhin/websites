TypeScript

Установка

Установка TypeScript

npm install -g typescript

Создание tsconfig.json

tsc -init

    // tsconfig.json:
  
{
  "compilerOptions": {
    "target": "es5",
    "module": "commonjs",
    "strict": true,
    "esModuleInterop": true
  }
}

TS позволяет использовать несколько tsconfig.json файлов в структуре каталога.

Компиляция

tsc

Нотация типов

name_1 : type_1;

name_2 : type_2 = value_2;
    // Types:

: string
: number
: boolean
: number []
: string []
: any

Типизация с поддержкой вывода типов - даже если не указать тип, компилятор проверит, где переменная была впервые использована, а затем использует этот тип для оставшейся части блока кода.
Для объектов используется утиная типизация

Перечисления

enum Name_1 {
  Name_11,
  Name_12,
  Name_13
}


    // Можно устанавливать значения вручную:

enum Name_2 {
  Name_21 = 11,
  Name_22 = 22,
  Name_23 = 33
}


    // Можно устанавливать строковые значения:

enum Name_3 {
  Name_31 = "string_1",
  Name_32 = "string_2",
  Name_33 = "string_3"
}
