Компонент React

React Component можно создавать с помощью функций и с помощью классов.
React Component может возвращать JSX, null, boolean, массив объекта, Fragment, Портал и с 16-го React можно возвращать sting и number. 

class Name_1 extends React.Component {
  render() {
    return (
      jsx_code_1
    );
  }
}

ReactDOM.render(
  <Name_1/>,
  document.querySelector('css_selector_1')
);

Name_1 - имя компонента, с большой буквы
