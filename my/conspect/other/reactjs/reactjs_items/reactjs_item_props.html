Properties

props

Задаются при вызове компонента и используются для изменения генерируемых элементов.
props не изменяемые.
Атрибуты задаваемые компоненту при его вызове, попадают внутрь него и называются props, и доступ к его значению получается через this.props.атрибут
Имена свойств соответствующие именам стандартных атрибутов HTML-элементов рендерятся, а нестандартные нет.
Чтобы отрендерить свойство с нестандартным именем, нужно записать его data-"property_1", где property - не стандартное имя свойства. 

function Name_1(props) {
  return (
    <React.Fragment>
      <tagName_11> {props.name_1} </tagName_11>
      <tagName_12> {props.name_2} </tagName_12>
      ....
      <tagName_1n> {props.name_n} </tagName_1n>
    </React.Fragment>
  );
}

import React from 'react';

  return (
