Раскрывающиеся объекты

Спойлер

    HTML:

<div class="spoiler">
    <input type="checkbox" id="sptoggler">
    <label for="sptoggler">
        <span>
        </span>
        ..."Header"...
    </label>
    <div>
        ..."Content"...
    </div>
</div>


    CSS:

.spoler {
    ...
}
.spoiler > input {
    display: none;
}
.spoler > label {
    display: block;
    ...
}
.spoler > div {
    display: none;
    ...
}
.spoler > label > span {
    font-family: monospace;
}
.spoler > label> span:before {
    content: "[+] ";
}
.spoler > input:checked ~ div {
    display: block;
}
.spoler > input:checked ~ label > span:before {
    content: "[-] ";
}

<span> - служит для вывода индикатора состояния
input {display:none;} - скрывает флажок
.a ~ .b - любой элемент .b , которому предшествует элемент .a

Меню-гамбургер

    HTML:

<div class="hamburger">
    <input type="checkbox" id="hambtoggler">
    <label for="hambtoggler">Button</label>
    <nav>
        <a href="#">One</a>
        <a href="#">Two</a>
        <a href="#">Three</a>
    </nav>
</div>


    CSS:

.hamburger {
    ...
}
.hamburger > input {
    display: none;
}
.hamburger > label {
    display: block;
    ...
}
.hamburger > nav {
    display: none;
    ...
}
.hamburger > nav > a {
    display: block;
}
.hamburger > input:checked ~ nav {
    display: inline-block;
}

Аккордеон 2

    HTML:

<div class="accordion">
    <div class="accordion-item">
        <input class="accordion-item__input" type="checkbox" id="accordion-1">
        <label class="accordion-item__trigger" for="accordion-1">
            Trigger 1
        </label>
        <div class="accordion-item__content">
            Content 1
        </div>
    </div>

    <div class="accordion-item">
        <input class="accordion-item__input" type="checkbox" id="accordion-2">
        <label class="accordion-item__trigger" for="accordion-2">
            Trigger 2
        </label>
        <div class="accordion-item__content">
            Content 2
        </div>
    </div>
</div>


    CSS:

.accordion {}

.accordion-item {
    margin-bottom: 20px;
    position: relative;
}

.accordion-item__input {
    position: absolute;
    top: 0;
    left: 0;
    opacity: 0;
}

.accordion-item__trigger {
    padding: 20px;
    border: 1px solid black;
}

.accordion-item__content {
    padding: 20px;
    border: 1px solid black;
    border-top: 0;
    display: none;
}

.accordion-item__input:checked ~ .accordion-item__content {
    display: block;
}
