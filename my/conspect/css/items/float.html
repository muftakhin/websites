Float

• Свойство float позволяет поместить элемент, позиционированный с помощью схемы нормального потока, с левого или с правого края элемента-контейнера
• Всё, что находится внутри элемента-контейнера, будет обтекать плавающий элемент
• При использовании свойства (float) должна быть задана ширина (width) плавающего элемента
• Если в элементе-контейнере всем элементам задать своство float, то некоторые браузеры будут воспринимать будто элемент-контейнер имеет высоту в 0px 

float: left;
     - right;
     - none;     /* default */
     - inherit;

• Если задать inline элементу float: left; или float: right;, он будет вести себя как block элемент
• Если в элементе-контейнере всем элементам задать своство float, то некоторые браузеры будут воспринимать будто элемент-контейнер имеет высоту в 0px

Обтекание inline элементов

• Необходимо задавать ширину (width) обтекаемого inline элемента
• Обтекаемые inline элементы ведут себя как блочные (поля со всех сторон сохраняются)
• Поля (margin) обтекаемых элементов не схлопываются

Обтекание блочных элементов

• Необходимо задавать ширину (width) обтекаемого блочного элемента
• Обтекаемый элемент остаётся ниже любых блочных элементов предшествовавших ему в исходном коде

Схлопывание контейнера

Если в элементе-контейнере всем элементам задать своство float, то некоторые браузеры будут воспринимать будто элемент-контейнер имеет высоту в 0px

Варанты решения:

Сделать контейнер плавающим

#container {
    float: left;
        ...
}

Воспользоваться свойством overflow:

#container {
    overflow: auto;
            - hidden;
    width: 100%;
        ...
}

Воспользоваться псевдоэлементом :after

#container:after {
    content: "";
    display: block;
    clear: both;
}

Запрет обтекания

Свойство clear позволяет указать, что ни один элемент, находящийся в том же элементе-контейнере, не должен касаться левой, правой или обеих стороны блока

clear: left;
     - right;
     - both;
     - none;     /* default */
     - inherit;

Ширина float элементов заданная в процентах, может по разному расчитываться разными браузерами. В данный момент использование не желательно
