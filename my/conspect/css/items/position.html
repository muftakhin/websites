Position

Типы позиционирования

position: static;    /* default */
        - relative;
        - absolute;
        - fixed;
        - inherit;
        - sticky;

Static

Обычное размещение в потоке

Relative

• Сдвигает блок относительно его начального положения в потоке
• Исходное пространство в потоке сохраняется и продолжает влиять на расположение окружающего контента
• Потенциально может перекрыть другие элементы

Absolute

• Элемент удаляется из потока и позиционируются относительно ближайшего родительского элемента не position: static;
• По умолчанию задано top: auto; right: auto; bottom: auto; left: auto;, если не задавать top, right, bottom или left, то элемент будучи вырванным из потока, окажется над тем же местом, которое он освободил
• Пространство, которое он занимал бы, смыкается
• Абсолютно позиционированные элементы не влияют на расположение окружающих элементов
• Если задать противоположные расстояния, то элемент растянется

• Элемент помещается относительно его ближайшего содержащего элемента
• html - начальный содержащий блок
• Абсолютно позиционированные объекты всегда ведут себя как блочные

Fixed

• Элемент удаляется из потока и позиционируется относительно окна браузера
• Смещение всегда относительно окна браузера

Sticky

top, right, bottom, left указывают границу, дальше которой элемент не будет перемещаться

Невозможно, когда высота позиционированного элемента равна высоте родительского элемента или значение его свойства overflow равна не visible

Все типы позиционирования (position:) ,кроме static делают inline элемент блочным (display: block;)

Задавание положения

Положение задаётся с помощью 4х свойств смещения

position: ... ;
top:
right:
bottom:
left: px;
    - %;
    - auto;       /* default */
    - inherit;


/* Применяется к позиционированным элементам (relative, absolute, fixed) */

Содержащие блоки

• Если позиционированный элемент содержится внутри другого не позиционированного элемента, тогда он будет размещён относительно начального содержащего блока (html)
• Если элемент содержится внутри другого элемента позиционированного (relative, absolute, fixed), он будет позиционирован относительно краёв этого элемента
• Позиционированные элементы становятся содержащим блоком для элементов, которые он содержит

Перекрывающиеся объекты

Порядок наложения

• z-index применяется элементам позиционированным (relative, absolute или fixed)
• По умолчанию элементы имеют значение свойства z-index: auto; и располагаются в том порядке,в котором появляются в документе (исходя из положения в HTML коде). Родительский элемент располагается ниже дочернего, а из дочерних более низко описаный в коде элемент располагается выше на странице 

z-index: число (целое);
       - auto;          /* default */
       - inherit;

z-index - число, чем больше значение, тем выше располагается элемент (ближе к переднему плану),дочерние элементы наследуют z-index родителя. При равном значении z-index на передний план выходит тот элемент, который в коде HTML располагается ниже
• Родительский элемент не может располагаться выше дочернего элемента, какой бы z-index мы ему не присвоили. При этом значение z-index образует для вложенных элементов свой уровень иерархии внутри данного уровня. Так потомки одного элемента с указанным значением z-index никогда не будут выше потомков другого элемента с указанным более высоким значением z-index
