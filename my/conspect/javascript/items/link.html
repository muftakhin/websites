Подключение

Ссылкой на отдельный файл

Внутри <head>

<head>
    ...
    <script src="name_1.js"></script>
    ...
</head>

Внутри <body>

<body>
    ...
    <script src="name_1.js"></script>
</body>

При этом код между тегами script игнорируется, если он есть

Как встроенный элемент

Внутри <head>

<head>
    ...
    <script>
        ...
    </script>
    ...
</head>

или внутри <body>

<body>
    ...
    <script>
        ...
    </script>
</body>

• Чтобы заработала интерактивность после загрузки всей страницы, JavaScript подключают в самом низу
• Атрибут defer, который указывает, что сценарий не будет изменять структуру страницы, а потому его можно безопасно выполнить после синтаксического анализа всей страницы

Для экранирования символа применяют символ \
