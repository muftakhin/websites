Подключение

Способы подключения

Синхронное подключение/загрузка

<body>
  ....
  <script src="script.js">
  </script>
</body>

    // Или:

<body>
  ....
  <script>
....
  </script>
</body>

Асинхронное подключение/загрузка

Загрузка не блокирует дальнейшее выполнение кода.

async

Выполняется сразу после скачивания. 

<body>
  <script src="script.js" async >
  </script>
</body>

defer

Выполняется когда DOM сформирован, но до события DOMContentLoaded.
Отложенные скрипты сохраняют порядок. 

<body>
  <script src="script.js" defer >
  </script>
</body>
