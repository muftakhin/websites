Дата

Конструкторо new Date()

Дата в JavaScript измеряется в милисекундах от 1 января 1970г. 

let nowDate = new Date();   // текущая дата

Передаётся в виде: гггг-мм-ддTчч:мм:сс:мммZ 

    // Можно передать информацию в нескольких вариантах:

let nowDate = new Date('2022-07-22');

let nowDate = new Date(2022, 7, 22, 1, 35);   // вместо 7 выведется 8 ибо месяц считается с 0
                                              // , а вместо 1 выведется время с учётом часового пояса

Получение компонентов Date()

let nowDate = new Date();

nowDate.getFullYear();        // год
nowDate.getMonth();           // месяц
nowDate.getDate();            // день
nowDate.getHours();           // час
nowDate.getMinutes();         // минуты
nowDate.getSeconds();         // секунды
nowDate.getMilliseconds();    // милисекунды
nowDate.getDay();             // день недели (с воскресенья)
nowDate.getTime();            // время в милисекундах
