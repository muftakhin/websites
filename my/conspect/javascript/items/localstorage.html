Local Storage

Объект, уникальный для каждого домена.
Браузеры выделяют объём 10Мб.

Методы localStorage

.setItem()

Создание новой записи. 

localStorage.setItem('name', value);

    // name - имя ключа
    // value - значение ключа

Если ключ с таким именем уже существует, то его значение перепишется.

.getItem()

Получение записи. 

localStorage.getItem('name');

    // name - имя ключа

.removeItem()

Удаление записи из хранилища. 

localStorage.removeItem('name');

    // name - имя ключа

clear()

- метод полной очистки хранилища
