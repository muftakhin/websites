Регулярные выражения

Введение

Реализованы объектом RegExp и интегрированы в методы строк.
Регулярные выражения состоят из шаблона и флагов. 

regexp = new RegExp("pattern", "flags");    // Длинная запись.

regexp = /pattern/;                         // Без флагов.
regexp = /pattern/gmi;                      // С gmi флагами.
    // Pattern:

. - все символы (как *)
\ - экранирование спецсимволов
\d - числа
\D - не числа
\w - буквы, слова
\W - не буквы, не слова
\s - пробелы


    // Flags:

i - независимо от регистра
g - все совпадения
m - многострочный режим
s - режим dotall (точка равна переводу строки)
u - вкл. полная поддержка юникода, разрешается крректная обработка суррогатных пар
y - поиск на конкретной позиции в тексте

Методы

.search()

let name_1 = name_2.search(name_3);

    // name_2 - строка
    // name_3 - регулярное выражение

Ищет только первое совпадение.

.match()

let name_1 = name_2.match(name_3);

    // name_2 - строка
    // name_3 - регулярное выражение

Если есть флаг g возвращает массив с соспадениями.
Если его нет, возвращает массив с первым совпадением и свойствами с дополнительной информацией.
Если нет совпадений - пустой массив.

.replace()

Заменяет совпадения. 

let name_1 = name_2.replace(name_3, "string_1");

    // name_2 - строка
    // name_3 - регулярное выражение
    // string_1 - на что заменяем

Если есть флаг g возвращает массив с соспадениями.
Если его нет, возвращает массив с первым совпадением и свойствами с дополнительной информацией.
Если нет совпадений - пустой массив.

.test()

let name_1 = name_3.test(name_2);

    // name_2 - строка
    // name_3 - регулярное выражение

Возвращает true/false в зависимости есть/нет совпадений.
