Условия и циклы

Условные конструкции

Условными конструкциями можно пользоваться, чтобы запускать код в нужные моменты, а циклами - чтобы программа выполнялась так долго, как требуется.

Конструкция if

if (contition) {    // Тело оператора if выполняется, только когда условие даёт true
  action;
}

Конструкция if ... else

if (condition) {
  action_1;
} else {
  action_2;
}

Цепочка конструкций if ... else

if (condition_1) {
  action_1;
} else if (condition_2) {
  action_2;
} else if (condition_3) {
  action_3;
} else {
  action_4;
}

Указывать завершающий else не обязательно.

switch

switch использует строгое сравнение без приведения типов. 

let name_1 = value;

switch (name_1) {
case value_2:
  action_1;
  break;
case value_3:
  action_2;
  break;
....
case value_n:
  action_n;
  break;
default:        // не обязательно
  action_x;
  break;
}
switch (condition) {
case true:
  action_1;
  break;
case false:
  action_2;
  break;
}

Тернарный оператор ? :

(condition) ? action_if_true : action_if_false;

let name = (condition) ? value_if_condition_true : value_if_condition_false;

Циклы

Цикл while

Цикл с предусловием. 

while (condition) {     // выполняется пока condition = true
  action;
  step;
}

Цикл do while

Цикл с постусловием.
В отличие от while обязательно выполняет первую итерацию. 

do {
  action;
  step;
} while (condition);    // условие проверяется после выполнения

Цикл for

for (start_point; condition; step) {
  action;
}

start_point - стартовое значение переменной счётчика цикла. Обычно это объявление и инициализация переменной.
condition - проверяемое условие, определяющее продолжение цикла.
step - изменение стартового значения. Часто это i++. 

for (; condition; ) {   // Не обязательно заполнять все 3 секции
  action;
}

Цикл for, массивы и строки

    // Пербор элементов массива:

let name_1 = ["one", "two", "three", "four"];
for (let i=0; i < name_1.length; i++) {
  console.log("name_1(i)");
}
    // Перебор символо строки:

let name_1 = "string_1";
for (let i=0; i < name_1.length; i++) {
  console.log("name_1(i)");
}

Другие варианты применения цикла for

Не обязательно сначала задавать переменной цикла значение 0, а затем каждый раз увеличивать её на 1.

Пример: Вывести все степени цифры 2 не превышающие числа 10000. 

for (let x=2; x < 10000; x=x*2) {
  console.log(x);
}

Прерывание цикла

for (start_point; condition_1; step) {
  action;

  if(condition_2) {
    break;
  }
}

Пропуск итерации

for (start_point; condition_1; step) {
  if(condition_2) {
    continue;
  }

  action;
}
