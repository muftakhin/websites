Все типы данных, кроме объектов называются примитивами

Примитивные типы данных передаются по значению, а объекты и т.п. по ссылке.

Просмотр типа данных

typeof name_1;

name_1 - переменная

typeof NaN;    //-> number
typeof null;   //-> object

Преобразование типов данных

Преобразование в число

name_1 = '123';
name_1 *= 1;

---

parseInt(name_1);    // в целое
parseFloat(name_1);  // в с плавающей точкой

---

Number(name_1);       // если name_1 - строка из цифр

---

+name_1;              // унарный плюс

name_1 - имя переменной содержащей исходные данные

Преобразование в строку

name_1 = 123;
name_1 += '';           // конкатенация (сложение чего-либо со трокой)

---

name_2 = name_1.toString();

---

name_2 = String(name_1);

Преобразование из 16 и 8

let name_1 = parseInt('value_1', 16);
let name_1 = parseInt('value_1', 8);

Преобразование в bollean

!!"string_1";
