Просмотр типа данных

typeof name_1;

Возвращает строку.
name_1 - переменная. 

typeof NaN;    //-> number
typeof null;   //-> object

Преобразование типов данных

Преобразование в число

Происходят при арифметических операциях.

*=

name_1 *= "value_1";

parseInt()

В целое число. 

parseInt(name_1);

parseFloat()

В число с плавающей точкой. 

parseFloat(name_1);

Number()

Строку из цифр в число.
Если нельзя преобразовть в число (содержит буквы), то вернёт NaN. 

Number(name_1);

  //

  underined -> NaN
  null      -> 0
  true      -> 1
  false     -> 0

Унарные + и -

+name_1;
-name_1;

Преобразование в строку

Сложение со строкой

name_1 + "string";

name_1 += "";

.toString()

name_1.toString();

String()

String(name_1);

Преобразование из 16 и 8

let name_1 = parseInt('value_1', 16);
let name_1 = parseInt('value_1', 8);

Преобразование в boolean

Boolean()

Boolean(name_1);

    //  Все true, кроме:

  0         ->  false
  ''        ->  false
  NaN       ->  false
  undefined ->  false
  null      ->  false
!!"string_1";
