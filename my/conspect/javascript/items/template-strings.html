Template string

`${...}`

Выражение внутри вычисляются и при необходимости преобразуются в строку

• Альтернатива конкатенации строк
• Позволяют вставлять в строки переменные (в тексте, например)
• Внутри шаблонной строки сохраняются пробелы и возможны операции
• Возможна вложенность одной шаблонной строки в другую 

`... ..., ... ${template_string_1}, ${template_string_2} ... ${template_string_n} ...`

template_string - любой код JS возвращающий значение
... - символы и слова строки
