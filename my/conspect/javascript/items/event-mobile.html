touchmove

Перемещение при касании элемента.

touchend

При прекращении касания элемента.

touchenter

При перемещении при касании над элементом.

touchleave

При перемещении при касании за пределы элемента.

touchcancel

Точка прикосновения не регистрируется в браузере.
Палец вышел за пределы браузера (например при серфинге)
