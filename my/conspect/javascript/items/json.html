JSON

Объект браузера JSON

Браузеры имеют встроенный объект JSON.

Методы объекта JSON

JSON.stringify()

Преобразует JavaScript-объект в JSON. 

const name_1 = {
  name_11: value_11,
  name_12: value_12,
  .... ,
  name_1n: value_1n
};

.... JSON.stringify(name_1) .... ;

JSON.parse()

Преобразует JSON в JavaScript-объект. 

JSON.parse(....);
    // Глубокое клонирование объектов при помощи stringify() и parse():

const name_1 = JSON.parse(JSON.stringify(name_2));
