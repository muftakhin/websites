Ассоциация

Лучше весь текст в форме обрамлять label-ом

<label> ... </label>

Неявная ассоциация

В элемент label внедряются элементы формы вместе с его описанием

<ul>
    <li> <label> <input type="checkbox" name="icecream" value="vanilla"> ванильное </label></li>
    <li> <label> <input type="checkbox" name="icecream" value="cherry"> вишнёвое </label></li>
    <li> <label> <input type="checkbox" name="icecream" value="chokolate"> шоколадное </label></li>
</ul>

Явная ассоциация

Сопоставляется метка с идентификатором id элемента формы

<label for="one"> Дядя: </label>
<input type="text" name="ancles_name" id="one"> <br>
<label for="two"> Тётя: </label>
<input type="text" name="aunts_name" id="two">
