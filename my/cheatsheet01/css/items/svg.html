SVG

Добавление в HTML

Используя тег <img>

• Подходит для контентных изображений
• Встроенные стили работают
• Скрипты не работают 

<img src='data:image/svg+xml;utf8,<svg>...<svg>'>
<img src='data:image/svg+xml;base64,PHN2ZyB4SD623FK...'>

INLINE SVG в <img>

• Не кешируется
• "Замусоривает" разметку
• Дубликаты утяжеляют страницу 

<img src="images/name.svg" alt="svg picture" width="..." height="...: ⁄>

Используя тег <object> , <embed> или <iframe>

• Работают встроенные стили и скрипты
• Можно управлять из JavaScript

<iframe>

• Неверно с идеологической точки зрения
• Не подстраивается под размер содержимого 

<iframe src="pic.svg" width="400" height="400">
<⁄iframe>

<object>

Позволяет встраивать не-HTML* содержимое в HTML-документ
* - PDF, FLASH, SVG 

<object data="images/name.svg" type="image/svg+xml" width="200" height="200">
<⁄object>

Необходимо указание хотябы одного из атрибутов

<embed>

Позволяет встраивать не-HTML содержимое в HTML-документ
* - PDF, FLASH, SVG 

<embed src="images/name.svg" width="200" height="200">

Необходимо указание хотябы одного из атрибутов

Непосредственная вставка

• Не кешируется
• "Замусоривает" разметку
• Дубликаты утяжеляют страницу 

<div class="look_it_this">
   <svg>
       ...
   <⁄svg>
<⁄div>

Добавление в CSS

INLINE SVG в CSS

• Кешируется вместе с CSS-файлом
• Ухудшает читаемость кода 

.background {
    background-image: url('data:image/svg+xml;utf8,<svg>...<svg>');
}

.background {
    background-image: url('data:image/svg+xml;base64,PHN2ZyB4SD623FK...');
}

В качестве фонового изображения

• Подходит для оформительских изображений
• Встроенные стили работают
• Скрипты не работают 

width: ...;
background-repeat: no-repeat;
background-image: url('images/name.svg');
background-size: 200px auto;
