SVG Фильтры

filter: blur();
      - brightness();
      - contrast();
      - drop-shadow();
      - grayscale();
      - hue-rotate();
      - invert();
      - opacity();
      - saturate();
      - sepia();
      - url();
      - none;         (default)
      - initial;
      - inherit;

url() - ссылка на SVG-фильтр в этом или другом документе

Фильтры можно объединять через пробел 

filter: blur() opacity() sepia();
blur - Размытие
blur(значение);
filter: blur(16px);
            (1em);

• значение задаётся в единицах длины (px, em, ...)
• По умолччанию равно 0

brightness - Яркость
brightness(значение);
filter: brightness(70%);
                  (0.7);

• значение задаётся в % или в десятичых дробях
• По умолччанию равно 1
• значение > 100% увеличивает яркость

contrast - Контрастность
contrast(значение);
filter: contrast(80%);
                (0.8);

• значение задаётся в % или в десятичых дробях
• По умолччанию равно 100%
• значение > 100% усиливает констрастность

drop-shadow - Тень
drop-shadow(x y размытость растяжение цвет);
filter: drop-shadow(16px 32px 8px -4px black);
                   (1em 2em 0.5em -0.25em #000);

Добавляется к элементам и его содержимому с учетом их прозрачности

grayscale - Оттенки серого
grayscale(значение);
filter: grayscale(40%);
                 (0.4);

• значение задаётся в % или в десятичых дробях
• По умолчанию 0% т.е. нет эффекта
• При 100% полностью обесцвечивается

hue-rotate - Оттенок
hue-rotate(0deg ... 360deg);

• Меняет цвета изображения в зависимости от заданного угла поворота в цветовом круге
• По умолчанию 0deg т.е. нет эффекта

invert - Инвертация
invert(0% ... 100%);

• По умолчанию 0% т.е. нет эффекта

opacity - Прозрачность
opacity(100% ... 0%);

• 0% - полностью прозрачный
• По умолчанию 100% т.е. нет эффекта

saturate - Насыщенность
saturate(значение);

• значение указывается числом или процентами
• 0% убирает цветность
• По умолчанию 100% т.е. нет эффекта и равно 1
• значение > 100% усиливает насыщенность

sepia - Сепия
sepia(0% ... 100%);

• По умолчанию 0% т.е. нет эффекта
