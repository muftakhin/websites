Переключатели и флажки

Переключатель

<input type="radio">
<input type="radio" name="..." value="..." cheked="cheked">

name="..." - объединяет несколько переключателей в группу, одно для всех вариантов
cheked="cheked" - вариант по умолчанию
value=".." - значение отправляемое на сервер вместе с именем выбранного элемента
