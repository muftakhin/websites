Атрибуты

Количество строк

rows="..."

Количество символов по ширине

cols="..."

Максимальное число символов

maxlength="..."

wrap

wrap="..."

Атрибут wrap говорит браузеру, как осуществлять перенос текста в поле <textarea> и в каком виде отправлять данные на сервер

soft - не сохраняет переносы строк (по умолчанию)
hard - сохраняет

Отключение изменение размера поля в CSS

textarea {
    resize: none;
}
