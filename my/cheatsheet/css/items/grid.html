Grid

Grid-контейнер

.containerName {
    display: grid;
           - inline-grid;
}

• Элемент с { display: grid; } в отличии от элемента с { display: block; } не "заползает" под float-элемент
• Отступы grid-контейнеров не схлопываются с отступами дочерних элементов
• Для grid-контейнеров игнорируются свойства семейства column
• Для grid-элементов игнорируются псевдоэлементы ::first-line и ::first-letter и не обрабатываются свойства float и clear , а свойство vertical-align работает для содержимого, а не для них самих

Размер колонок

.containerName {
    grid-template-columns: width_1 width_2 ... width_n;
}

• width - ширина колонки в (px , fr , % или em и смешение их). Где fr - доля колонки от свободного пространства (как flex-grow:)
• Размер можно задавать как min-content (минимальное расстояние) 

.containerName {
    grid-template-columns: repeat(auto-fill, minmax(240px, 1fr));
}

• auto-fill- заполнить пространство максимально возможным числом колонок, с минимальным или максимальным размером
• auto-fit -

minmax(размер_1, размер_2) где:
размер_1 - минимальный размер, до которого максимально ужмётся, когда все остальные максимально ужмутся
размер_2 - максимальный размер, пока остальные не ужмутся польностью

Размер строк

.containerName {
    grid-template-rows: heigth_1 heigth_2 ... heigth_n;
}

.containerName {
    grid-template-rows: repeat("количество повторений", "повторяющаяя последовательность");
}

heigth - высота строки в (px , fr , % или em и смешение их). Где fr - доля строки (как flex-grow:)
auto - полоса будет автоматически увеличиваться по мере необходимости, в соответствии с размером контента
repeat() - функция - сокращённый вариант для определения нескольких полос
min-content - по контенту 

grid-template-rows: repeat(4, auto);

    эквивалентно:

grid-template-rows: auto auto auto auto;
grid-template-rows: repeat(3, 2fr 1fr);

    эквивалентно:

grid-template-rows: 2fr 1fr 2fr 1fr 2fr 1fr;
grid-template-rows: 1fr repeat(2, 3fr) 1fr;

    эквивалентно:

grid-template-rows: 1fr 3fr 3fr 1fr;

Gap

Позволяет не использовать отрицательные margin по краям контейнера

Зазор между строками

grid-row-gap: значение;

    новее:

row-gap: значение;

Зазор между колонок

grid-column-gap: значение;

    новее:

column-gap: значение;

Зазор между ячейками

grid-gap: значение;
        - row-gap / column-gap;

Grid-элемент

Если элемент занимает одну ячейку сетки

.itemName {
    grid-column: "номер занимаемого столбца";
    grid-row: "номер занимаемой строки"
}

Если элемент занимает более одной ячейки сетки

Первый вариант записи

.itemName {
    grid-column-start: "номер_верт.линии_от";
    grid-column-end: / "номер_верт.линии_до";
    grid-row-start: "номер_гор.линии_от";
    grid-row-end: "номер_гор.линии_до";
}

Вариант сокращённой записи:

.itemName {
    grid-column: "номер_верт.линии_от" / "номер_верт.линии_до";
    grid-row: "номер_гор.линии_от" / "номер_гор.линии_до";
}

• Минус перед номером меняет направление отсчёта: 1 - первая линия с начала
-1 - первая линия с конца

Второй вариант записи

.itemName {
    grid-column: "номер_верт.линии_от" / span "кол-во_занимаемых_ячеек_колонки";
    grid-row: "номер_гор.линии_от" / span "кол-во_занимаемых_ячеек_строки";
}

Третий вариант записи

.itemName {
    grid-area: "номер_гор.линии_от" / "номер_верт.линии_от" / "номер_гор.линии_до" /"номер_верт.линии_до";
}

Или:

.itemName {
    grid-area: "номер_гор.линии_от" / "номер_верт.линии_от" / span "кол-во_занимаемых_ячеек_строки" / span "кол-во_занимаемых_ячеек_колонки";
}

Выравнивание

Выравнивание внутри контейнера

Горизонтальное выравнивание

justify-content: start;      /* default */
               - end;
               - center;
               - stretch;
               - space-around;
               - space-between;
               - space-evenly;

Вертикальное выравнивание

align-content: start;       /* default */
             - end;
             - center:
             - stretch;
             - space-around;
             - space-between;
             - space-evenly;

Горизонтальное и вертикальное выравнивание

place-content: "align-content" / "justify-content";

Выравнивание внутри элемента

Горизонтальное выравнивание

justify-items: start;     /* default */
             - end;
             - center;
             - stretch;

Вертикальное выравнивание

align-items: start;    /* default */
           - end;
           - center:
           - stretch;

Горизонтальное и вертикальное выравнивание

place-items: "align-items" / "justify-items";

Выравнивание отдельной ячейки

.itemName {
    justify-self: start;   /* default */
                - end;
                - center;
                - stretch;

    align-self: start;   /* default */
              - end;
              - center:
              - stretch;

    place-self: align-self / justify-self;
              - ...;
}

Именованные линии сетки

.containerName {
    grid-template-columns: [имя_перв.линии] ширина_1_кол. [имя_ср.линии] ширина_2_кол. [имя_посл.линии];
}
.itemName {
    grid-column: "имя от" / "имя до";
}

• Можно задать линии несколько имён через пробел, и использовать любое из них
• Можно задать одинаковые имена с разными "суффиксами" и обращаться через одно имя сразу к двум именам, между которыми находится полоса 

.containerName {
    grid-template-columns: [start] 1fr [center] 2fr [end];
}
.itemName {
    grid-column: start / center;
}
.containerName {
    grid-template-columns: [left-start] 1fr [left-end right-start] 2fr [right-end];
}
.itemName {
    grid-column: left;
}

• Можно задать одинаковые имена с разными "суффиксами" и обращаться через одно имя сразу к двум именам, между которыми находится полоса

Именованные области сетки

.itemName-1 {
    grid-area: name-1;
}
.itemName-2 {
    grid-area: name-2;
}
...
.itemName-n {
    grid-area: name-n;
}

.containerName {
    grid-template-columns: ... ;
    grid-template-areas: ... ;
    grid-template-areas: "name name ..."
                         "name ...  ..."
                         "...  ...  ...";
}

Автоматическая генерация строк и колонок

.containerName {
    ...
    grid-auto-rows: "размер";
    grid-auto-columns: "размер";
}

•

grid-auto-flow

.containerName {
    ...
    grid-auto-flow: row;          /* default */
                  - column;
                  - row dense;
                  - column dense;
                  - dense;
}

• dense - автоматически уплотняет
