Селекторы

Селекторы элементов

Универсальный селектор

*

Применяется ко всем элементам в документе

Селектор элемента (тега / типа)

element

Соответствует имени элемента

Сгруппированные селекторы

element_1, element_2, ..., element_n

Дочерний селектор

parent_element>child_element

Соответсвует элементу, являющемуся прямым потомком другого элемента (дочернему элементу)

Селектор потомка

element_1 element_2

Соответствует элементу, являющемуся потомком (не обязательно прямым) другого элемента (потомку любого уровня вложенности)

Селектор соседнего элемента

element_1 + element_2

Селектор родственного элемента

element_1 ~ element_2

Селектор идентификатора

#Name

Селектор класса

.ClassName

Селекторы псевдокласса

Псевдоклассы ведут себя как дополнительные значения атрибута class="" к которым можно применять различные стили

Якорные псевдоклассы

Псевдоклассы :link и :visited позволяют применять различные стили к ещё не посещённым и посещённым ссылкам

:link
:visited

Псевдоклассы действий пользователя

Позволяют изменять внешний вид элементов в ответ на действия пользователя

:focus
:hover
:active
element_1:hover element_2 {
...: ...;
}

Применится к element_2 при наведении на element_1

Дочерний селектор

:first-child
:last-child
:only-child

:first-child - первый
:last-child - последний
:only-child - единственный 

:nth-child(...)
       odd  - нечётные
       even - чётные
       2n
       2n+1
       число
       ...

Дочерний селектор по типам

:first-of-type
:last-of-type
:nth-of-type

Селектор пустого элемента

:empty
.item:empty {
display: none;
}

Селекторы ещё

:target
:enabled
:disabled

Селектор отрицания

selector_1:not(selector_2)

- для всех элементов selector_1, кроме элемента selector_2

ul:hover li:not(:hover)

Селекторы :: псевдоэлементов

Первая строка или буква

Псевдоэлементы действуют как дополнительный элемент в коде. При использовании псевдоэлементов :first-line и :first-letter вокруг первой строки текста или первой буквы как-будто создаётся дополнительный элемент, к которому можно применять собственные стили 

::first-line {
    color:
    font:
    background:
    word-spacing:
    letter-spacing:
    text-decoration:
    vertical-align:
    text-transform:
    line-height:
}

::first-letter {
    color:
    font:
    background:
    word-spacing:
    letter-spacing:
    text-decoration:
    text-transform:
    vertical-align:
    padding:
    margin:
    line-height:
    border:
    float:
}

Генерируемый контент

::before {
    content: "...";
}

::after {
    content: "...";
}

Псевдоэлемент по умолчанию display: inline;, ему можно задать display: block; с content: ""; и он станет вести себя как полноценный блок

Выделенный контент

::selection {
    color: "...";
    background-color: "...";
}

Можно изменить лишь свойства color: и background-color:

Placeholder

::placeholder {
...
}

Можно изменить лишь свойства color: и background-color:

Селекторы атрибутов

[ ] - Селектор простых атрибутов (существования атрибута)

Соответствует конкретному атрибуту независимо от его значения

элемент[атрибут]
img[title] {border: 3px solid;}

[ = ] - Селектор точного значения атрибута (равенства атрибута)

Соответствует конкретному атрибуту с конкретным значением

элемент[атрибут="точное значение"]
img[title="first grade" {border: 3px solid;}

[ ~= ] - Селектор неполного значения атрибута (атрибута с пробелами)

Сответствует конкретному атрибуту, чьё значение присутствует в списке слов, разделённых пробелами (позволяет задать часть значения атрибута отделённую пробелами)

элемент[атрибут~="часть значения отделённая пробелами"]
img[title~="grade"] {border: 3px solid;}

[ |= ] - Селектор разделённого дефисом значения атрибута

Позволяет задать часть значения атрибута, отделённую дефисами

элемент[атрибут|="часть-значения-отделённая-дефисами"]
*[hreflang|="en"] {border: 3px solid;}

[ ^= ] - Селектор начала значения атрибута (префикса атрибута)

Соответствует конкретному атрибуту, чьё значение начинается с конкретной строки

элемент[атрибут^="первая часть значения"]
img[src^="/images/icons"] {border: 3px solid;}

[ $= ] - Селектор конца значения атрибута (суффикса атрибута)

Соответствует конкретному атрибуту, чьё значение заканчивается конкретной строкой

элемент[атрибут$="последняя часть значения"]
a[href$=".pdf"] {border: 3px solid;}

[ *= ] - Селектор части значения в любом месте атрибута (подстроки атрибута)

Соответствует конкретному атрибуту, чьё значение содержит конкретную подстроку

элемент[атрибут*="любая часть значения"]
