Форматирование текста

Подключение шрифтов

Свойство @font-face позволяет использовать шрифт, даже если он не установлен на компьютере пользователя, предоставляя путь к файлу шрифта 

@font-face {
    font-family: Name;
    src: url{".../..."};
}

.class_name {
    font-family: Name, ..., ...;
}
другой вариант написания присваиваемого имени шрифта:

@font-face {
    font-family: "Name";
    src: url{".../..."};
}

.class_name {
    font-family: "Name", ..., ...;
}

• font-family: присваивает шрифту по данному url имя Name 

@font-face {
    font-family: ChunkFiveRegular;
    src: url{"fonts/chunkfive.eot"};
}
h1, h2 {
    font-family: ChunkFiveRegular, Georgia, serif;
}

Свойства шрифта

Семейство шрифтов (Гарнитура шрифта)

• Свойство font-family даёт возможность задать список предпочтительных шрифтов
• Если имя шрифта состоит не из одного слова, то его указываем в двойных кавычках
• В конце списка всегда указываем название семейства шрифтов(serif, sans-serif, cursive или monospace)
• Вместо названия семейства шрифтов браузер будет применять тот шрифт, который определён им как используемый по умолчанию для указанного семейства шрифтов 

font-family: "Trebuchet MS", Verdana, sans-serif;

Размер шрифта

Конкретным размером

font-size: ...px;
         - ...em;
         - ...ex;
         - ...rem;

em - размер относительно заданного или наследуемого размера шрифта данного элемента
rem - размер относительно шрифта в корневом элементе (<html>)

Процентным значением

font-size: ...%;

Используя абсолютное зарезервированное слово

font-size: xx-small;
         - x-small;
         - small;
         - medium;
         - large;
         - x-large;
         - xx-large;

Используя относительное (сравнительное) зарезервированное слово

font-size: larger;
         - smaller;

Насыщенность шрифта

font-weight: normal;
           - bold;
           - bolder;
           - lighter;
           - 100;
           - 200;
           - 300;
           - 400;
           - 500;
           - 600;
           - 700;
           - 800;
           - 900;
           - inherit;

Начертание шрифта

font-style: normal;
          - italic;
          - oblique;
          - inherit;

oblique - наклонный шрифт

Капитель

font-variant: normal;
            - small-caps;
            - inherit

Сокращённая запись

font: font-style font-weight font-variant font-size/line-height font-family;

Выравнивание строк текста

Интерлиньяж

Интерлиньяж - минимальное (ибо рисунок в строке его увеличит) расстояние между базовыми линиями строк текста.
Базовая линия - воображаемая линия, на которой располагаются нижние части символов. 

line-height: число (масштабный множитель на текущий размер;
           - ...em;
           - ...%;
           - normal;
           - inherit;

Можно воспользоваться свойством line-height, чтобы выровнять текст по средней линии блока: 

text-align: center;
line-height: (высота блока);

Красная строка

text-indent: ...px;
           - ...em; ()
           - ...% (на основе ширины родительского элемента);

Применяется к блочным элементам и ячейкам таблицы

Горизонтальное выравнивание

text-align: left;
          - right;
          - center;
          - justify;
          - inherit;

justify определяет, что каждая строка абзаца, кроме последней, должна занимать всю ширину текстового поля.
Применяется к блочным элементам и ячейкам таблицы

Вертикальное выравнивание

vertical-align: baseline;
              - sub;
              - super;
              - top;
              - text-top;
              - middle;
              - bottom;
              - text-bottom;
              - px;
              - em;
              - %;

vertical-align выравнивает только inline-элементы и содержимое ячеек таблицы

Декоративные эффекты

text-decoration: none;
               - underline;
               - overline;
               - line-throught;
               - blink (мерцание);
               - inherit;

Изменение регистра

text-transform: none;
              - capitalize (первая буква каждого слова заглавная);
              - lowercase;
              - uppercase;
              - inherit;

Интервал между знаками (кернинг) и интервал между словами

letter-spacing:                 - между знаками

word-spacing: значение длины;   - между словами
            - normal;
            - inherit;

Обработка пробелов между словами и переносы строк внутри элемента

white-space: normal;     (default)
           - nowrap;
           - pre;
           - pre-wrap;
           - pre-line;
           - break-spaces;
           - initial;
           - inherit;

nowrap - отмена переносов, кроме <br>
pre-wrap - сохранение пробелов, делая разрыв строк, где необходимо
pre-line - удаление лишних пробелов, за исключением случая с <br>

Тень текста

text-shadow: "смещение по горизонтали" "смещение по вертикали" "радиус размытия" "цвет";

• Если размытие не требуется его можно опустить
• Можно применить несколько теней, через запятую 

text-shadow: .2em .2em .15em silver;
text-shadow: 8px 8px black;
text-shadow: .2em .2em .15em silver, 8px 8px 16px black;

Изменение маркеров и нумерации списков

Выбор маркера

Маркеры находятся по умолчанию вне элемента <li>, слева, в padding-left: элемента <ul> 

list-style-type: none;
               - disc;        (default)
               - circle;
               - square;
               - decimal;
               - decimal-leading-zero;
               - lower-alpha;
               - upper-alpha;
               - lower-latin;
               - upper-latin;
               - lower-roman;
               - upper-roman;
               - lower-greek;
               - upper-greek;
               - inherit;

Положение маркера

list-style-position: inside;
                   - outside;
                   - inherit;
                   - handing;

Создание пользовательских маркеров

list-style-image: url('images/pic.gif');
                - none;
                - inherit;
