Цвета и фон

Основной цвет

color: значение (имя/число);
     - inherit;

Фоновый цвет

background-color: значение (имя/число);
                - transparent;          (default)
                - inherit;

Непрозрачность

opacity: число от 0 до 1;        (по умолчанию - 1)

0 - прозрачный
1 - не прозрачный
Наследуется и не изменяется во вложенных элементах

Фоновые изображения

Добавление фонового изображения

background-image: url(images/image.gif);
                - none;                  (default)
                - inherit;

• Если не задан padding:, фон заходит под border:
• Если задать вместе background-color: и background-image:,изображение будет помещено поверх цвета

Управление мозаичным покрытием фона

background-repeat: repeat;   (default)
                 - repeat-x;
                 - repeat-y;
                 - no-repeat;
                 - inherit;

Положение фона

background-position: "значение_X" "значение_Y";      (default: 0% 0% /(left top))
                   - left;
                   - center;
                   - right;
                   - top;
                   - bottom;
                   - inherit;
                   - top left;
                   - top center;
                   - top right;
                   - left center;
                   - center center;
                   - right center;
                   - bottom left;
                   - bottom center;
                   - bottom right;
                   - "направление" "значение"

По умолчанию background-position отсчитывается от padding-box, чтобы border не закрывал собой фоновое изображение

Позиционирование фона

background-origin: content-box;
                 - padding-box;  (default)
                 - border-box;

Можно задавать через запятую несколько систем отсчёта (для разных фоновых изображений)

Фиксация фона

background-attachment: scroll;   (default)
                     - fixed;
                     - inherit;
                     - local; (перемещение внутри элемента)

Размер фона

background-size: auto;   (default)
               - ширина высота;
               - %;
               - cover;
               - contain;
               - initial;
               - inherit;

auto - естественный размер элемента
cover - заполняет, сохраняя соотношение сторон
contain - вписывает, сохраняя соотношение сторон

Сокращённое свойство фона

background: background-color background-image/url("images/image.gif")
            background-repeat background-attachment background-position
            background-size;

          - inherit;

• Порядок значений не важен
• В background-image URL записывается в кавычках, в отличии от отдельного указания background-image

Множественные фоновые изображения

background-image: url(image_1.png), url(image_2.png), url(image_3.png);
background-position: left top, center center, right bottom;
background-repeat: no-repeat; no-repeat; no-repeat;
            
background:
    url(image1.png) left top no-repeat,
    url(image2.png) center center no-repeat,
    url(image3.png) right bottom no-repeat;

background-clip

background-clip: border-box;     (default)
               - padding-box;
               - content-box;

border-box - фон обрезается внешней границей рамки
padding-box - фон обрезаеится внешней границей padding (внутренней границей border)
content-box - фон обрезается границей контента

Градиенты

Линейные градиенты

linear-gradient(color_1, color_2);

- простейший вариант 

linear-gradient(direction, color_1, color_2);
            
direction = to top
            to right
            to bottom
            to left
            to top right
            to top left
            to bottom right
            to bottom left
            ...deg
#banner {
    background-image: linear-gradient(180deg, yellow, green);
}
            
    или
            
background-image: linear-gradient(to bottom, yellow, green);
linear-gradient(direction, color_1 dot_1, ... ..., color_n dot_n);
            
    dot = % пути по направлению (может быть со знаком " - ")
#banner {
    background-image: linear-gradient(90deg, yellow, orange 25%, blue);
}

- оранжевый появляется через 25% пути по линии градиента 

linear-gradient(yellow 50%, lightblue 50%);

Если точки перехода цвета совпадают, то один цвет резко сменит другой в этой точке перехода 

linear-gradient(yellow 50%, lightblue 0);

Если задать следующую точку равную 0, то она автоматически будет в том же месте, что и предыдущая (если меньше предыдущей, то на том же месте, что и предыдущая)

Радиальные градиенты

radial-gradient(size type direction, color_1, ..., color_n)
#banner {
    background-image: radial-gradient(center, contain, yellow, green);
}
