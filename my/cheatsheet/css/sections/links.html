Способы подключения

Встроенные (inline) стили

<p style="font-size: 20px"> Текст <⁄p>

Внутренние таблицы стилей

<head>
    ...
    <style>
        p {
            font-size: 20px;
        }
    <⁄style>
    ...
<⁄head>

Внешние таблицы стилей

Использованием элемента link

<head>
    ...
    <link rel="stylesheet" href="style.css">
    ...
<⁄head>

• Подключать внешние стили желательно выше в документе
• При использовании нескольких таблиц стилей при помощи элемента <link>, внутри элемента <head> помещают отдельный элемент <link> для каждой таблицы стилей
• Если более одного правила применяются к одному и тому же элементу, то указанный

Импортирование

@import url("styles⁄style.css");

• @import Добавляется во внешний файл .css либо в элемент style
• При использовании правила @import в таблице стилей, его необходимо помещать перед всеми остальными правилами
• @import ухудшают загрузку

Модульные таблицы стилей

⁄* настройки форматирования текста *⁄
@import url("type.css")
⁄* обработка форм *⁄
@import url("forms.css")
⁄* навигация *⁄
@import url("list-nav.css")
⁄* конкретные стили *⁄
body {background: orange;}
...

Указываются перед правилами, которые используют селекторы
