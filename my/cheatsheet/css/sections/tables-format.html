Форматирование таблиц

Расположение заголовка

caption-side: top;     (default)
            - bottom;
            - initial;
            - inherit;

initial устанавливает значение по умолчанию

Промежуток между ячейками

border-collapse: separate;
               - collapse;
               - initial;
               - inherit;
border-spacing: расстояние;
              - "расстояние по X" "расстояние по Y";
              - initial;
              - inherit;

Видимость пустых ячеек

empty-cells: show;
           - hide;
           - initial;
           - inherit;

Компоновка макета

table-layout: auto;
            - fixed;
            - initial;
            - inherit;
