Блочная модель

Задавание размеров блока

• margin-"сторона": auto; занимает всё свободное пространство в этом направлении
• Если задать ширину блока, то margin, border и padding будут добавлены к его ширине и высоте 

width:
height:
min-width:
max-width:
min-height:
max-width: ...px;
         - ...%;
         - auto;
         - inherit;
         - min-content;
         - max-content;
         - fit-content;
         - stretch;

• Применяется к block элементам и изображениям
• inline элементам width и height не задаются (не работают)
• % задаются от ближайшего родительского элемента с заданным аналогичным размером (если ни у одного из предков не задана ширина, отсчитывается от ширины окна браузера)

Модель border-box

* {

Красивые границы

border-image: border-image-source border-image-slice border-image-width border-image-outset border-image-repeat;

border-image-source - URL изображения
border-image-slice - значения 4х линий разреза по часовой
border-image-width - ширина границы
border-image-outset - расстояние, на которое изображение выступает за границы
border-image-repeat - образ заполнения сторон (stretch, repeat, round) 

border-image: url(border.png) 55 55 55 55 stretch;

55 - border-image-width - ширина от каждого края изображения, по часовой (top, right, bottom, left) 

border-width: 55px;
border-image: url(border.png) 55 stretch;

Контур

• outline отрисовывается за пределами границы элемента иможет перекрывать другое содержимое (не является частью размеров элемента) • Отрисовывается вокруг элементов находящихся в фокусе

Стиль контура

outline-style: dotted;
             - dashed;
             - solid;
             - double;
             - groove;
             - ridge;
             - inset;
             - outset;
             - none;
             - hidden;

Цвет контура

outline-color: значение;

Ширина контура

outline-width: значение;

Сокращённая запись

outline: "outline-style" "outline-color" "outline-width";

Смещение контура

outline-offset: значение;

Может быть отрицательным

Поля

margin: значение;
      - %;
      - auto;      (default)
      - inherit;

% - процент от размера ближайшего родительского элемента с указанными размерами (так же как и padding)

Поведение полей

• Верхние и нижние поля соседних элементов схлопываются (смежные поля совмещаются и используется большее значение). Кроме float: ... и position: absolute
• Левое и правое никогда не схлопываются
• Если у родительских элементов не указаны border:, padding: или overflow:hidden; (то, что ограничивало бы), то вертикальные margin: перетекают к родительскому элементу (выпадают), вплоть до < body > (если при этом у родительского элемента задано значение margin: , то выбирается наибольшее из них)
• Только левое и правое поля отображаются у inline элементов, кроме изображений
• Значение может быть отрицательным

Типы отображения

display: inline;
       - block;
       - list-item;
       - inline-block;
       - table;
       - inline-table;
       - table-row-group;
       - table-header-group;
       - table-footer-group;
       - table-row;
       - table-column-group;
       - table-column;
       - table-cell;
       - table-caption;
       - none;
       - run-in;
       - compact;
       - ruby;
       - ruby-base;
       - ruby-base-container;
       - ruby-text-container;
       - flex;
       - grid;

• none полностью удаляет контент из потока, не сохраняя пространство. В отличие от visibility: hidden; которое делает элемент невидимым сохраняя пространство
• inline-block заставляет блочный элемент располагаться как встроенный, но при этом сохранять остальные особенности блочного. Чтобы блок не "плясал" (из-за значения по умолчанию vertical-align: baseline; ), нужно задать явно vertical-align: center, vertical-align: top или завернуть элементы в родительский блок той же высоты). По умолчанию <img> имеет значение inline-block
• list-item создаёт маркер у элемента (при помощи list-style-type задаётся тип маркера)

Сокрытие блоков

Свойство visibility позволяет скрывать блоки, но при этом оставлять пустое пространство на месте элемента 

visibility: hidden;
          - visible;

Тень блока

box-shadow: x y b e c z;
          - none;            (default)

x - горизонтальное смещение
y - вертикальное смещение
b - размытие (может отсутствовать)
e - распространение - увеличивает или уменьшает размер тени
c - цвет
z - тень внутрь " inset " (не обязательный параметр)

• Можно применять сразу несколько теней, через запятую
• Если первые три параметра будут иметь нулевое значение, тень будет иметь вид рамки
