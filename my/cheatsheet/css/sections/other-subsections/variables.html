var()

Переменные или пользовательские свойства

• Каскадируются и наследуются
• Ведут себя как переменные с областью видимости 

    Объявление:
            
:root {
    --имя-переменной: значение;
}
            
    Вызов:
            
.class {
    свойство: var(--имя-переменной);

• С целью хранения пользовательского свойства (переменной) в корне документа используем псевдокласс :root
• Переменные можно переназначать в потомках
• Переменную можно задать в элементе (локально) и использовать в потомках этого элемента

Добавление резервного значения

Функция var() принимает второй параметр, который задаёт резервное значение 

    Объявление:
            
:root {
    --имя-переменной: значение_1;
}
                            
    Вызов:
                            
.class {
    свойство: var(--имя-переменной, значение_2);
    Объявление:
                
:root {
    --имя-переменной_1: значение_1, значение_2;
    --имя-переменной_2: значение_3;
    --имя-переменной_3: значение_4;
}
                                
    Вызов:
                                
.class {
    свойство_1: var(--имя-переменной_1, значение_5);
    свойство_2: var(--имя-переменной_2, --имя-переменной_3);
