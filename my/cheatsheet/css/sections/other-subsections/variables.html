Переменные

    Объявление:
            
:root {
    --имя: значение;
}
            
    Вызов:
            
.class {
    свойство: var(--имя);

• С целью хранения пользовательского свойства (переменной) в корне документа используем псевдокласс :root
• Переменные можно переназначать в потомках
• Переменную можно задать в элементе (локально) и использовать в потомках этого элемента
