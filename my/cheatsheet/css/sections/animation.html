Переходы, преобразования и анимация

CSS переходы

Основные сведения о переходах

• transition-property        - свойство
• transition-duration        - продолжительность перехода
• transition-timing-function - ускорение перехода
• transition-delay           - пауза перед началом

Определение свойств

transition-property: свойство;
                   - all;        (default)
                   - none;

Применяется:
  - ко всем элементам
  - к псевдоэлементам :before и :after

Продолжительность перехода

transition-duration: 0s;     (default)

Применяется:
  - ко всем элементам
  - к псевдоэлементам :before и :after

Функция тайминга

transition-timing-function: ease;        (default)
                          - linear;
                          - ease-in;
                          - ease-out;
                          - ease-in-out;
                          - step-start;
                          - step-end;
                          - steps;
                          - cubic-bezier(#,#,#,#)

Применяется:
  - ко всем элементам
  - к псевдоэлементам :before и :after

Задержка

transition-delay: 0s;

Применяется:
  - ко всем элементам
  - к псевдоэлементам :before и :after

Сокращённое свойство перехода

transition: свойство продолжительность тайминг задержка;

Применение множественных переходов

transition-property: background-color, color, letter-spacing;
transition-duration: 0.3s, 2s, 0.3s;
transition-timing-function: ease-out, ease-in, ease-out;
transition: background-color 0.3s ease-out, color 2s ease-in, letter-spacing 0.3s ease-out;

Переход на все случаи

transition: all 0.2s ease-in-out;

CSS-преобразования

transform: функция(и) преобразования
         - none                          (default)

Преобразование угла (поворот)

transform: rotate(...deg);

Смещение центра вращения

transform-origin: смещение_по_X смещение_по_Y;

смещение задаётся в %;
                    px;
                    left;
                    center;
                    right;
                    top;
                    bottom;

по умолчанию: 50% 50%;
img {
    width: 300px;
    height: 400px;
    transform: rotate(-10deg);
    transform-origin: center top;
                    = 50%, 0%;
                    = 150px, 0px;
}

Преобразование позиции (перемещение)

transform: translateX(смещение_по_X);

transform: translateY(смещение_по_Y);


transform: translate(смещение_по_X, смещение_по_Y);

    если одно значение, то X

Задаётся в px или %

Преобразование размера (масштабирование)

transform: scaleX(значение);

transform: scaleY(значение);

transform: scale(значение_X, значение_Y);

transform: scale(значение_X_и_Y);

Если значение будет со знаком минус (-), то элемент перевернётся по данной оси

Скашивание (наклон)

transform: skewX(...deg);

transform: skewY(...deg);

transform: skew(...deg, ...deg);

Перспектива

transform: perspective(...px);

Точка перспективы

transform: perspective-origin: ...% ...%;    (defaulf: 50% 50%;)

Множественные преобразования

transform: функция(значение) функция(значение); 

.class:hover {
    transform: scale(1.5) rotate(-5deg) translate(50px, 30px);
}

Преобразования применяются в том порядке, в котором перечислены

Анимация

@keyframes "имя" {
    from {
        свойство: значение_1;
    }
    ...% {
        свойство: значение_2;
    }
    ...% {
        свойство: значение_3;
    }
    to {
        свойство: значение_n;
    }
}

element {
    animation-name: "имя";
}

name - имя анимации
from - начальный ключевой кадр
to - конечный ключевой кадр 

@keyframes color_font {
    from {
        color: black;
    }
    to {
        color: red;
    }
}

Продолжительность анимации

animation-duration: ...s;
                  - ...ms;

Функция тайминга

animation-timing-function: ease-in-out;

Как у transition:

Можно задать для каждого интервала 

@keyframes color_font {
    from {
        color: black;
        animation-timing-function: ease-in-out;
    }
    50% {
        color: green;
        animation-timing-function: linear;
    }
    to {
        color: red;
        animation-timing-function: ease-out;
    }
}

Задержка

animation-delay: ...s;
               - ...ms;

Как у transition:

Количество итераций

animation-iteration-count: число;
                         - infinite;

Направление

animation-direction: alternate;

Вид по окончании анимации

animation-fill-mode: forwards;

Сокращённая запись анимации

animation: "name" "duration" "timing-function" "iteration-count" "direction" "delay" "fill-mode";

Если к елементу применяется несколько анимаций, списки сокращённых свойств записываются через запятую 

element {
    animation-name: "имя_1", "имя_2", "имя_n";
    animation-duration: ...s, ...s, ...s;
}

Приостановка анимации

элемент:Псевдокласс {
    animation-play-state: paised;
                        - running;
}

Применить это свойство можно только с помощью псевдокласса
