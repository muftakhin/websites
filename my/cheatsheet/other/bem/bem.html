Б__Э_М

Блок

class="б  б_м"

б - блок
м - модификатор

• Выполняет одну функцию
• Состоит из элементов объединённых логически
• Допускается вложенность блоков друг в друга

Какие свойства стиля нельзя использовать для класса .блок

• margin:
• position: со свойствами указывающими координаты по направлениям ( top, right, bottom, left, ... )
• float:

Элемент

class="б__э  б__э_м"

э - элемент

- составная часть блока, которая не может быть и спользована без него

Модификатор

• Модификатор определяет внешний вид, состояние или поведение элемента
• Модификатор не используется самостоятельно 

class="б  б_м  б_м_з"
class="б__э  б__э_м_з"

з - значение

Нельзя использовать сразу 2 класса с "...модификатор_значение"

Селекторы и порядок их написания

1   .блок
   
2   .блок_модификатор
   
3   .блок__элемент
   
4   .блок__элемент_модификатор

Если элемент является блоком

    .блок__элемент элемент
