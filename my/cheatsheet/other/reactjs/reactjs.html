React JS

Элемент React

Создание элемента React

React.createElement(tag-name_1, propertys_1, content_1);

Где:
tag-name_1 - "тип элемента"
propertys_1 - свойства (если нет, то null)(если атрибуты, то {name_1: "value_1, name_2: "value_2"} )
content_1 - то, что между тегами (дочерние элементы/содержимое)

Создание списка:

React.createElement(
    "section",
    {id: "container"},
    React.createElement(
        "ul",
        {"className": "list"},
        React.createElement("li", {"className": "list__item"}, "string_1"),
        React.createElement("li", {"className": "list__item"}, "string_2"),
        React.createElement("li", {"className": "list__item"}, "string_3"),
    )
)

Создание списка из элементов массива name_1:

React.createElement(
    "ul",
    {"className": "list"},
    name_1.map(
        item => React.createElement("li", {"className": "list__item"}, item)
    )
)

Отобразить элемент React в DOM

var name_1 = React.createElement(type, propertys, between);

ReactDOM.render(name_1, document.getElementById('react-container'));

Где:
name_1 - элемент, который необходимо отобразить
document.getElementById('react-container') - элемент с if="react-container" внутрь которого поместится элемент name_1

Создание элемента при помощи createFactory

React.DOM.tag-name_1(propertys_1, content_1)

Где:
tag-name_1 - "тип элемента"
propertys_1 - свойства (если нет, то null)(если атрибуты, то {name_1: "value_1, name_2: "value_2"} )
content_1 - то, что между тегами (дочерние элементы/содержимое)
