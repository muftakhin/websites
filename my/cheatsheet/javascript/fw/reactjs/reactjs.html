Основы

• Компонента - функция, возвращающая разметку JSX
• Мы "вставляем" компоненту как тег с именем соответствующим имени функции
• В JSX разметке внутри функции, все элементы должны быть обёрнуты в единый элемент 

const Name_1 = () => {
    return (
        ... );
}

Импорт компонентов

- для использования компонентов, находящихся в других файлах 

    name.js:

import Name_1 from '.⁄name_2.js';
import Name_3 from '.⁄..⁄name_4.js';

let name_5 = () => {

    <Name_1⁄>
    <Name_3⁄>

}

• Путь относительный, и начинается с ./, .js можно не ставить
• При этом в файле из которого импортируются компоненты должен быть прописан export 

    name_2.js :

let name_1 = () => {
    ...
}

export default name_1;
    name_4.js :

let name_3 = () => {
    ...
}

export default name_3;

В name.js Name_1 и Name_3 можно писать любые имена (из-за того, что в name_2.js и name_4.js написано default ), но желательно соответствующие компонентам и с большой буквы
