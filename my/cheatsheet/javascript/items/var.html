Переменные и константы

Переменные

• имя переменной может состоять из букв, цифр, символов $ и _
• имя переменной не может начинаться с цифры

• переменная var видна до её объявления
• все объекты в булевом контексте являются true

Инициализация и присвоение значения

Инициализация (создание)

let name_1;

var name_2;

var name_3,

Присвоение значения

let name_1 = value_1;

name_1 = value_2;
var name_3 = value_3;

name_3 = value_4;

var name_5 = value_5,   // Объявление в одной инструкции
    name_6 = value_6,   // несколько переменных
    name_7 = value_7;

При присвоении нового значения или ранее созданной переменной, использование ключевого слова не обязательно

Области видимости переменных

Если имена глобальной и локальной переменных совпадают, то локальная переменная замещает внутри функции глобальную

Глобальные

Глобальные

Интерпретатор вытягивает все объявления переменных var в верхнюю часть своей области видимости

• Видны даже в дополнительных сценариях
• Существуют, пока существует страница (при перезагрузке страницы уничтожаются и создаются заново)

var или внутри функции без использования ключевого слова var (используемые без предварительного объявления)

• Видны даже в дополнительных сценариях
• Существуют, пока существует страница (при перезагрузке страницы уничтожаются и создаются заново)

Глобальные

Объявленные за пределами функций с использованием ключевого слова var или внутри функции без использования ключевого слова var (используемые без предварительного объявления)

• Видны даже в дополнительных сценариях
• Существуют, пока существует страница (при перезагрузке страницы уничтожаются и создаются заново)

Локальные

Объявленные внутри функции используя ключевое слово var

• Доступны только локально в самой функции
• Параметр переданный функции приобретают локальную область видимости, кроме массивов, передаваемых функции по ссылке. Если внести изменение в элемент массива, переданного в качестве параметра, изменятся элемент исходного массива
• Переменная цикла видна во всей функции, внутри которой объявлена
• Создаются при первом вызове функции и исчезают при завершении функции
• При этом при создании переменной внутри блока кода if-else и for её область видимости этими блоками не ограничиваются (фигурные скобки этих блоков не изолируют область видимости этой переменной)

let

• Переменная видимая внутри блока кода { }, что защищает значение глобальной переменной

var name_1 = value_1;
                    
                    
if (name_1) {
    var name_1 = value_2;
}
    ⁄⁄ Значение name_1 переопределится
                    
                    
                    
if (name_1) {
    let name_1 = value_3;
}
    ⁄⁄ Значение name_1 не переопределится


var i = value_4;

for (var i=0, i<value_5, i++) {
    action_1;
}
    ⁄⁄ Значение глобальной переменной i не переопределится


for (let i=0, i<value_5, i++) {
    action_1;
}
    ⁄⁄ Значение глобальной переменной i переопределится

Константы

Записываются заглавными буквами

const NAME1;

• Нельзя присвоить новое значение
• Если значением является объект, то можно изменять его свойства

Удаление переменной

• Глобальные переменные созданные в глобальной области с помощью var не могут быть удалены
• Глобальные переменные созданные в локальной области без использоавания var (подразумаваемые глобальные переменные) могут быть удалены 

delete name_1;
