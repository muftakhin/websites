string

• Строка может быть заключена в одинарные или двойные кавычки
• Строка может содержать ноль и более символов
• \ является экранирующим символом
• Все символы являются 16-разрядными
• JavaScript не имеет символьного типа. Для того, чтобы получить символ, нужно задать строку из одного символа
• JS может временно преобразовывать строку-примитив в строку-объект
• Строка всегда примитив, если она не была создана с помощью конструкторов

Создание строки

let name_1 = "string";

• - cоздать строку name_1:
• string может быть пустым

Отдельный символ строки

name_1[0];     // первый символ строки name_1
name_1[1];     // второй символ строки name_1
. . . .
name_1[n];     // (n-1)-ый символ строки name_1

Методы строк

Конкатенация

let name_1 = "text_1";
let name_2 = "text_2";

let name_3 = name_1 + " text_3 " + name_2;

//or:

let name_3 = name_1.concat(" text_3" + name_2)

.length

- длина строки

name_1.length;

.slice()

- срез строки

name_1.slice(index_1, index_2);

• index_1, index_2 - позиции символов перед которыми производится срез строки name_1
• index_2 может быть отрицательным, тогда он отсчитывается с конца

.substr()

name_1.substr(start, length);

• index_1, index_2 - позиции символов перед которыми производится срез строки name_1
• index_2 может быть отрицательным, тогда он отсчитывается с конца

.substring()

- извлечение части строки

name_1.substring(i, j);

Возвратит часть строки начинающуюся с индекса i включая его, и заканчивающуюсю индексом j не включая его

name_1.substring(i);

Если не указывать второй индекс, метод возвратит часть строки начинающуюся с индекса i и до конца строки

.split()

- получает символ-ограничитель и разбивает строку на части по позиции символа-ограничителя и возвращает их в массиве

let name_2 = name_1.split("symbol");

symbol - символ, указывающий место разделения

let name_1 = "string_1|string_2|string_3";
let name_2 = name_1.split("|");

.toUpperCase()

- перевод строки name_1 в заглавный регистр

name_1.toUpperCase();

.toLowerCase()

- перевод строки name_1 в строчный регистр

name_1.toLowerCase();

.charAt(i)

- символ строки с индексом i

name_1.charAt(i);

.indexOf()

- индекс первого вхождения аргумента "text" в строке name_1

name_1.indexOf("text");
name_1.indexOf("text", i);

• i - индекс начальной позиции поиска
• Если аргумент в строке найти не удалось, метод возвратит -1

.lastIndexOf()

- индекс последнего вхождения аргумента "text" в строке name_1

name_1.lastIndexOf("text");
name_1.lastIndexOf("text", i);

• i - индекс начальной позиции поиска
• Если аргумент в строке найти не удалось, метод возвратит -1
