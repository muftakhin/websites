Модули

export

• Модуль- это фрагмент кода, пригодный для многократного использования
• Модули хранятся в отдельных файлах, по одному на модуль
• Из одного модуля можно экспортировать один или несколько объектов

Экспорт одного объекта (одну переменную) из файла file_name_1.js:

const name_1 = new Name_2(...);

export default name_1

Экспорт двух объектов (две функции) из файла file_name_2.js:

export const name_3(...) => action_3_1;

export const name_4(...) => {
    name_4_1: value_4_1;
    name_4_2: value_4_2() => action_4_1;
}

import

import name_1 from './file_name_1'

import name_3 from './file_name_2'
import name_4 from './file_name_2'

• В модулях с несколькими экспортируемыми переменными можно обратиться к деструктурированию объектов

import { name_4_1, name_4_2 } from './file_name_2';

• Для переменными модуля можно задать локальную область видимости под ругими именами переменных

import { name_4_1 as name_5, name_4_2 as name_6 } from './file_name_2';

• Можно импортировать все в одну переменную

import * as name_6 from './file_name_2';
