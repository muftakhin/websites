Список

<select>...<⁄select>

Может содержать два и более элементов <options>...<⁄options>

Пункт списка

<option>...<⁄option>

Используется для указания вариантов ответа (пунктов раскрывающегося списка)

Логическая группа пунктов списка

<optgroup>...<⁄optgroup>

Раскрывающийся список

Если элемент options указан без атрибута value, то в качестве значения будет отправляться содержимое элемента options (текст) 

<p> Ваш любимый исполнитель?
    <select name="singer">
        <option value="shura"> Шура <⁄option>
        <option value="prof"> Профессор Лебединский <⁄option>
        <option value="linda"> Линда <⁄option>
        <option value="abp"> Алла Пугачёва <⁄option>
    <⁄select>
<⁄p>

    <select name="actor" size="3" multiple>
        <option value="stallone"> Сильвестр Сталлоне <⁄option>
        <option value="arnold" selected="selected"> Арнольд Шварценеггер <⁄option>
        <option value="norris"> Чак Норрис <⁄option>
        <option value="jkvd"> Жан Клод Ван Дамм <⁄option>
        <option value="chan"> Джекки Чан <⁄option>
        <option value="lee"> Брюс Ли <⁄option>
    <⁄select>
<⁄p>

Ваш любимый актёр?

multiple - позволяет выбрать более одного элемента из списка
selected="selected" устанавливает вариант ответа, выбраный по умолчанию во время загрузуи страницы

Группировка пунктов раскрывающегося списка

<⁄p>
