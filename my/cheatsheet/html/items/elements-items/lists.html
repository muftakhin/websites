Cписок определений

Применяется для расшифровки терминов

<dl>...</dl>

• содержит только dt и dd
• dt может иметь несколько dd

Имя (термин/метка)

<dt> ... </dt>

содержит только строчные элементы

Значение (описание/определение)

<dd> ... </dd>

может содержать любой элемент потокового контента

<dl>
    <dt> ... </dt>
    <dd> ... </dd>
    <dt> ... </dt>
    <dd> ... </dd>
</dl>
